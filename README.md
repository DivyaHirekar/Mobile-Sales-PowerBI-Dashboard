# 📱 Mobile Sales Performance Dashboard

## Overview
This Power BI dashboard analyzes mobile sales performance across brands, cities, and payment methods.

## Tools Used
- Power BI
- Excel
- Data Analysis
- Data Visualization

## Key Metrics
- Total Sales: 769M
- Total Quantity: 19K
- Average Price: 40K
- Total Transactions: 4K

## Key Insights
- Apple is the top-selling brand.
- July recorded the highest sales.
- UPI is the most used payment method.
